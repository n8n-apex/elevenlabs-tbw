ElevenLabs AI Learning Suite Embed

Option 1: With Agent ID (Secure)

Use this when you want to embed with a pre-configured agent ID:

<script src="https://your-domain.com/embed-simple.js" data-agent-id="agent_3801k4sspqk2f5j8feh1x2g7x079" data-height="600px" data-width="100%" async> </script>

Option 2: User Input (Flexible)

Use this when you want users to input their own agent ID:

<script src="https://your-domain.com/embed-simple.js" data-height="600px" data-width="100%" async> </script>

Available Parameters

  • data-agent-id - Your ElevenLabs agent ID (optional)
  • data-height - Height of the embed (default: 600px)
  • data-width - Width of the embed (default: 100%)
  • data-container-id - ID of container element (optional)
  • data-base-url - Base URL if hosting elsewhere (optional)
